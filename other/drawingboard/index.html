Drawing Board

This Drawing Board is made using D3.

(Note that D3 is used instead of p5 because the developer is learning D3 and wanted to explore the capabilities of D3)

Since D3 uses SVG to create visuals, pixels objects are first created to simulate a canvas. We then place these pixels in their correct locations. Each pixels have OnClick and OnHover functions that allows the mouse click and mouse drag to change the style of the pixel which then simulates drawing.
