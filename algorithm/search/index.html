About Search Algorithm

Search algorithms are used to search and retrieve an element from a given data set of any data structure. It is one of the most basic algorithm concept but also one of the most popular and important functionality in a lot of software. Examples of Search algorithm are linear search and binary search

Linear Search

Linear search is the simplest algorithm for search algorithm. The algorithm is designed to look for the element one by one until the element is found or until the end of the data set Since the algorithm run through the whole element, the algorithm's BigO notation is O(n)

Binary Search

Binary search searches through the data set by dividing the list by half then checks the value of the middle element to know if the algorithm shouls either retrieve the element, move to the right, or move to the left. Due to the nature of how the algorithm is designed, the algorithm will only work properly if the data set is sorted based on how the Binary Search Algorithm is design(eg. if the binary case decision go to the left if the element the algorithm is trying to look for is smaller, then the data set should be sorted in ascending order)
